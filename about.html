About the project

Build-A-Politician

What This Is

Build-A-Politician is a fictional political strategy simulator. Every candidate you build or select is an invented character — no real names, likenesses, or endorsements are used anywhere in the project. It's meant purely as an entertaining way to explore how personal traits, policy platforms, and campaign events might interact in a simulated election.

The entire experience runs in your browser. There is no account, no server, and no data ever leaves your device — candidates you save are stored locally using your browser's storage.

Created By

Van Club America
Van, Texas

Build-A-Politician was designed and built as a self-contained, static web project — plain HTML, CSS, and JavaScript, with no frameworks or backend — so it can be hosted for free on GitHub Pages.

A Note on Fiction

This site is satire and entertainment. Some of the pre-built opponents in the roster are loosely inspired, in tone and era, by figures from American political history — but every name, party, and biographical detail has been invented for the game. Nothing on this site should be read as a factual claim about any real person.
